# Auto-Insurance-Claims-Analysis

In this project, we have a dataset that includes details of the insurance policy along with customer information. It also contains data about the accident on which the claims were made.

The dataset approximately consists of 1,000 rows and 40 columns, with column names such as policy number, policy bind date, annual premium, incident severity, incident location, auto model, and more.

The goal is simply to us python tools to analyse that dataset.

This dataset is here : "https://www.kaggle.com/datasets/buntyshah/auto-insurance-claims-data/code?datasetId=45152&sortBy=commentCount"
